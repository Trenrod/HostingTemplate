# Hosting template

Simple and reliable hosting template.

## Feature roadmap

### Version X.0
- [ ] Multiple servers support
- [ ] Different roles per server support
- [ ] Configurable support for:
	- [ ] Prometheus (metrics)
	- [ ] Loki (logs)

### Version 1.0
- [ ] Customized configuration generated overa a cli
- [ ] Automatic Certificates and renewal
- [ ] TLS (1.3 only)

## Requirements

- SSH access
- DNS A record
- Python
- NodeJS

## Tech stack

- Ansible
- Let`s Encrypt
- Envoy
- Acme.sh
- NodeJS Typescript
